# 「合肥大炮」

这是我在大一（2017 秋季）时力热的大作业。「合肥大炮」是温晓辉老师的「力学与热学」课程中非常知名的项目系列，要求将一颗炮弹从合肥以恰当的参数投向指定的地点（如~~紫金港~~）。

今天听到消息，老师打算今年把项目换成行星发动机。于是……应该可以放出来了吧。

仅作存档。
